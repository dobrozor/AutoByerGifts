# AutoByerGifts
Авто-скупщик подарков телеграмм. Без авторизации и прочего

## Настройка
Вставьте свой бот токен, можно и админ айди, но эьто на потом. Запускаешь бота, в боте пополняешь звезды, настраиваешь на скупку и бот работает 24\7 мониторит новые подарки, а если на балансе бота есть звезды, он будет отправлять вам подарки по вашим настройкам
